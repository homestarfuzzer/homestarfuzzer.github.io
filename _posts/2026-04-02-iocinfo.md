Working in a SOC means you look up a lot of indicators. IPs, domains, hashes all day. For a while my workflow was the same as everyone else's: open VirusTotal in one tab, AbuseIPDB in another, maybe iplocation in a third, copy/paste the IOC into each one, and piece the picture together yourself.

It works. But it's slow, and after doing it a few hundred times it starts to feel like something a script should be handling.

So I wrote one.

## What it does

`iocinfo` takes an IP, domain, or hash and queries multiple threat intel sources at once, then outputs everything in a single color coded view. Green means clean. Red means danger ahead.

```bash
iocinfo 8.8.8.8
iocinfo evil-domain.com
iocinfo d41d8cd98f00b204e9800998ecf8427e
```

It auto detects the indicator type, so you don't have to specify anything. Just run it.

You can also target specific sources if you only care about one or two:

```bash
iocinfo 1.2.3.4 --source abuseipdb
iocinfo 1.2.3.4 --source vt abuseipdb shodan
iocinfo evil.com --source vt threatfox urlhaus
```

Run `iocinfo --help` for the full source list with color coded free vs. paid tiers.

## Free vs. Paid Sources

One thing I wanted to get right was making sure the tool is useful even without API keys. A lot of tools like this are basically dead on arrival without paid accounts. This one isn't.

Out of the box with zero configuration, `iocinfo` queries:

- **ip-api.com**: geolocation, ASN, ISP, proxy and hosting flags
- **ipinfo.io**: hostname, org, abuse contact
- **ThreatFox**: malware family, confidence score, tags
- **URLhaus**: malware URLs, file signatures, download history
- **DNS**: live resolution for domains

None of those require a key. Install it and it works.

If you add API keys, you also get:

- **VirusTotal**: detection ratio across 90+ AV engines
- **AbuseIPDB**: confidence score and full report history
- **Shodan**: open ports, banners, CVEs
- **GreyNoise**: internet noise classification
- **AlienVault OTX**: pulse count and threat actor tagging

All of those have free tiers. VirusTotal gives you 500 lookups a day, AbuseIPDB gives you 1,000. More than enough for personal use.

## Setup

Install with pipx (recommended; this keeps it isolated from your system Python):

```bash
pip install pipx
pipx install git+https://github.com/homestarfuzzer/iocinfo.git
```

Then run the setup wizard to configure your API keys:

```bash
iocinfo --setup
```

It walks you through each source, shows where to sign up for a free key, and saves everything to `~/.iocinfo/config.ini`. Skip any source you don't want, the free sources work with no config at all.

## Output

Results are color coded based on the verdict:

- **Green**: clean, not flagged
- **Yellow**: low risk / worth noting
- **Orange**: moderate risk
- **Red**: high risk or confirmed malicious

VirusTotal coloring is based on detection ratio — 0/94 is green, 7/94 is red. AbuseIPDB is based on confidence score. The idea is you can scan the output in a couple seconds and know whether something needs a deeper look.

## Config reference

If you'd rather skip the wizard and edit directly, the config lives at `~/.iocinfo/config.ini`:

```ini
[virustotal]
api_key = YOUR_KEY_HERE

[abuseipdb]
api_key = YOUR_KEY_HERE

[shodan]
api_key = YOUR_KEY_HERE

[greynoise]
api_key = YOUR_KEY_HERE

[otx]
api_key = YOUR_KEY_HERE

[ipinfo]
token = YOUR_TOKEN_HERE
```

Leave out any section you don't have a key for. The tool skips unconfigured sources automatically.
